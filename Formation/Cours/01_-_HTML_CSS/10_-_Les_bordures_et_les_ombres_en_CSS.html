Les angles

Par défaut les angles d'un élément étant basés sur le pixel, ils sont carrés. De ce fait, les bordures suivent le même schéma.
Il nous est possible de casser ce comportement et d'arrondir les angles d'un élément.

Chaque angle peut être piloté indépendamment des autres.
