Les animations CSS

Les animations CSS rendent possible la création de transitions animées depuis une décoration CSS à une autre. Les animations consistent en deux composants : un style décrivant l'animation et un ensemble de keyframes qui indique les états de début et de fin du style CSS de l'animation, ainsi que des points de passage intermédiaires.

Tout comme pour les transitions, pour définir une nouvelle animation il est nécessaire de préciser au minimum :

  • la ou les propriété(s) à animer (dans le cas ci-dessous : la position par rapport au bord gauche)
  • la durée de l'animation (exprimée en seconde)

Ceci ne définit pas l'apparence réelle de l'animation, qui est elle définie par les règles @keyframes.


.div {
    background-color:lightblue;
    display:inline-block;
    height:150px;
    margin:15px;
    position:relative;
    width:150px;

    -webkit-animation-name:moveleft;
    -moz-animation-name:moveleft;
    -ms-animation-name:moveleft;
    -o-animation-name:moveleft;
    animation-name:moveleft; /* Défini le nom de la règle @keyframes décrivant les keyframes de l'animation. */

    -webkit-animation-duration:2s;
    -moz-animation-duration:2s;
    -ms-animation-duration:2s;
    -o-animation-duration:2s;
    animation-duration:2s; /* Configure la durée que l'animation doit utiliser pour compléter un cycle. */
}

@keyframes moveleft { /* Définition des règles à appliquer sur une échelle de temps (en pourcentage) */
    0% { /* équivaut à from {} */
        left:0;
    }
    100% { /* équivaut à to {} */
        left:400px;
    }
}

Nous pouvons aussi appliquer la transition à plusieurs propriétés en même temps (dans l'exemple ci-dessous, le background et le padding en même temps) :


.div {
    background-color:gray;

    height:150px;
    margin:15px;
    position:relative;
    width:150px;

    -webkit-animation-name:change-moveleft-bgcolor;

    -o-animation-name:change-moveleft-bgcolor;
    animation-name:change-moveleft-bgcolor;
    
    -webkit-animation-duration:2s;
    -moz-animation-duration:2s;
    -ms-animation-duration:2s;
    -o-animation-duration:2s;
    animation-duration:2s;
}

@keyframes change-moveleft-bgcolor {
    0% {
        background-color:lightblue;
        left:0;
    }
    100% {
        background-color:lightgreen;

    -ms-animation-name:change-moveleft-bgcolor;
    -o-animation-name:change-moveleft-bgcolor;

    -webkit-animation-duration:2s;
    -moz-animation-duration:2s;
    -ms-animation-duration:2s;

Le sens d'animation : animation-direction

Précise si l'animation doit se jouer à l'envers ou selon des cycles alternés.


.div {
    background-color:gray;
    display:inline-block;
    height:150px;
    margin:15px;
    position:relative;
    width:150px;

    -webkit-animation-name:change-moveleft-bgcolor;
    -moz-animation-name:change-moveleft-bgcolor;
    -ms-animation-name:change-moveleft-bgcolor;
    -o-animation-name:change-moveleft-bgcolor;
    animation-name:change-moveleft-bgcolor;

    -webkit-animation-duration:2s;
    -moz-animation-duration:2s;
    -ms-animation-duration:2s;
    -o-animation-duration:2s;
    animation-duration:2s;

    -webkit-animation-timing-function:ease-in-out;
    -moz-animation-timing-function:ease-in-out;
    -ms-animation-timing-function:ease-in-out;
    -o-animation-timing-function:ease-in-out;
    animation-timing-function:ease-in-out;

    -webkit-animation-iteration-count:infinite;
    -moz-animation-iteration-count:infinite;
    -ms-animation-iteration-count:infinite;
    -o-animation-iteration-count:infinite;
    animation-iteration-count:infinite;

    -webkit-animation-direction:alternate;
    -moz-animation-direction:alternate;
    -ms-animation-direction:alternate;
    -o-animation-direction:alternate;
    animation-direction:alternate; /* normal | alternate | reverse | alternate-reverse */

L'état d'animation : animation-play-state

Détermine si l'animation est en cours de lecture ou en pause. Vous pouvez interroger cette propriété afin de déterminer si l'animation est en cours de lecture ou non. De plus, vous pouvez modifier cette valeur pour mettre en pause ou reprendre la lecture de l'animation (par exemple au survol de la souris).


.div {
    background-color:gray;
    display:inline-block;
    height:150px;
    margin:15px;
    position:relative;
    width:150px;

    -webkit-animation-name:change-moveleft-bgcolor;
    -moz-animation-name:change-moveleft-bgcolor;
    -ms-animation-name:change-moveleft-bgcolor;
    -o-animation-name:change-moveleft-bgcolor;
    animation-name:change-moveleft-bgcolor;

    -webkit-animation-duration:2s;
    -moz-animation-duration:2s;
    -ms-animation-duration:2s;
    -o-animation-duration:2s;
    animation-duration:2s;

    -webkit-animation-timing-function:ease-in-out;
    -moz-animation-timing-function:ease-in-out;
    -ms-animation-timing-function:ease-in-out;
    -o-animation-timing-function:ease-in-out;
    animation-timing-function:ease-in-out;

    -webkit-animation-iteration-count:infinite;
    -moz-animation-iteration-count:infinite;
    -ms-animation-iteration-count:infinite;
    -o-animation-iteration-count:infinite;
    animation-iteration-count:infinite;

    -webkit-animation-direction:alternate;
    -moz-animation-direction:alternate;
    -ms-animation-direction:alternate;
    -o-animation-direction:alternate;
    animation-direction:alternate;
}
.div:hover {
    -webkit-animation-play-state:paused;
    -moz-animation-play-state:paused;
    -ms-animation-play-state:paused;
    -o-animation-play-state:paused;
    animation-play-state:paused; /* running | paused */
}

@keyframes change-moveleft-bgcolor {
    0% {
        background-color:lightblue;
        left:0;
    }
    100% {
        background-color:lightgreen;
        left:400px;
    }
}
