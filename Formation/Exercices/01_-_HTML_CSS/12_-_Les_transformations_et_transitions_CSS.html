Les transformations & transitions CSS - Mise en pratique

Suite au cours sur les transformations & transitions CSS, nous allons mettre en pratique les différentes propriétés CSS correspondant à ces instructions.
Vous allez trouver ci-dessous différens exercices vous permettant de les manipuler.

Ex. 1 : Ouvrir la porte

Pour cet exercice :

  • nous allons utiliser les transformations et les transitions pour ouvrir la porte (dans le bon sens).
  • le survol de la souris représentera le fait que nous frappions à la porte et nous attendrons 4 secondes avant que quelqu'un nous ouvre.
  • la porte mettra 1 seconde à s'ouvrir entièrement de manière linéaire.
  • la porte mettra 4 secondes à se refermer entièrement après un délai de 2 secondes le temps de passer.
  • le début de la fermeture se fera doucement puis accélèrera avant de ralentir sur la fin.
    information supplémentaire: il est possible d'indiquer des transitions différentes sur l'élément et sur son survol.

#ex1 {
    border:#000000 5px solid;
    height:250px;
    margin:15px auto;
    position:relative;
    width:90px;

    -webkit-perspective:800px;
    -moz-perspective:800px;
    -ms-perspective:800px;
    -o-perspective:800px;
    perspective:800px;
}
    #ex1 .porte {
        background-color:#c3c3c3;
        height:240px;
        position:relative;
        width:80px;
        z-index:10;

        -webkit-transform-style:preserve-3d;
        -moz-transform-style:preserve-3d;
        -ms-transform-style:preserve-3d;
        -o-transform-style:preserve-3d;
        transform-style:preserve-3d;

        -webkit-transform-origin:0 50%;
        -moz-transform-origin:0 50%;
        -ms-transform-origin:0 50%;
        -o-transform-origin:0 50%;
        transform-origin:0 50%;

        -webkit-transform:rotateY(360deg);
        -moz-transform:rotateY(360deg);
        -ms-transform:rotateY(360deg);
        -o-transform:rotateY(360deg);
        transform:rotateY(360deg);

        -webkit-transition:transform 4s ease-in-out 2s;
        -moz-transition:transform 4s ease-in-out 2s;
        -ms-transition:transform 4s ease-in-out 2s;
        -o-transition:transform 4s ease-in-out 2s;
        transition:transform 4s ease-in-out 2s;
    }
    #ex1 .porte:hover {
        -webkit-transform:rotateY(180deg);
        -moz-transform:rotateY(180deg);
        -ms-transform:rotateY(180deg);
        -o-transform:rotateY(180deg);
        transform:rotateY(180deg);

        -webkit-transition:transform 1s linear 4s;
        -moz-transition:transform 1s linear 4s;
        -ms-transition:transform 1s linear 4s;
        -o-transition:transform 1s linear 4s;
        transition:transform 1s linear 4s;
    }
        #ex1 .porte .poignee {
            background-color:#666666;
            height:10px;
            margin-top:-5px;
            position:absolute;
            right:20px;
            top:50%;
            width:10px;

            -webkit-border-radius:50%;
            -moz-border-radius:50%;
            -ms-border-radius:50%;
            -o-border-radius:50%;
            border-radius:50%;
        }
        #ex1 .porte .gond {
            background-color:#666666;
            height:10px;
            margin-top:-5px;
            position:absolute;
            left:-2px;
            width:4px;
        }
            #ex1 .porte #gondhaut {top:15%;}
            #ex1 .porte #gondbas {bottom:15%;}

</div>

Ex. 2 : Blocs d'informations

Pour cet exercice :

  • nous allons utiliser les transformations et les transitions pour afficher le bloc d'informations sur l'image survolée.
  • le bloc est caché en bas de l'image par défaut.
  • le bloc doit s'afficher en glissant vers le haut.
  • le bloc doit mettre 0.3 secondes à s'afficher.
  • le bloc ne faisant pas 100% de la hauteur de l'image, il doit rester aligné en bas de l'image.

    font-size:0;
    margin:15px auto;
    position:relative;

        width:30%;

        -moz-background-size:cover;

        -o-background-size:cover;
        background-size:cover;

        #ex2 .thumb-wrap .caption {

Ex. 3 : Card flip

Pour cet exercice :

  • nous allons utiliser les transformations et les transitions pour retourner la carte au survol.
  • la carte affiche par défaut la face 1 (id="front").
  • le bloc doit se retourner en fonction d'un axe placé en profondeur à 400px en retrait sur l'axe z puis revenir se plaquer sur l'écran.
  • une fois retournée, la carte doit afficher la face 2 (id="back").
  • les deux faces ne doivent pas s'afficher en même temps.
    information supplémentaire: il existe une instruction s'appelant "backface-visibility".

#ex3 {
    border:#cccccc thin solid;
    height:260px;
    margin:40px auto;
    position:relative;
    width:200px;

    -webkit-perspective:400px;
    -moz-perspective:400px;
    -ms-perspective:400px;
    -o-perspective:400px;
    perspective:400px;
}
    .card {
        cursor:pointer;
        height:100%;
        position:absolute;
        width:100%;

        -webkit-transform-style:preserve-3d;
        -moz-transform-style:preserve-3d;
        -ms-transform-style:preserve-3d;
        -o-transform-style:preserve-3d;
        transform-style:preserve-3d;

        -webkit-transform-origin:100% 50%;
        -moz-transform-origin:100% 50%;
        -ms-transform-origin:100% 50%;
        -o-transform-origin:100% 50%;
        transform-origin:100% 50%;
        
        -webkit-transition:transform 1s;
        -moz-transition:transform 1s;
        -o-transition:transform 1s;
        transition:transform 1s;
    }
    .card:hover {
        -webkit-transform:translateX(-100%) rotateY(-180deg);
        -moz-transform:translateX(-100%) rotateY(-180deg);
        -ms-transform:translateX(-100%) rotateY(-180deg);
        -o-transform:translateX(-100%) rotateY(-180deg);
        transform:translateX(-100%) rotateY(-180deg);
    }
        .card .face {
            display:block;
            color:#ffffff;
            font-size:140px;
            font-weight:bold;
            height:100%;
            line-height:260px;
            margin:0;
            position:absolute;
            text-align:center;
            width:100%;

            -webkit-backface-visibility:hidden;
            -moz-backface-visibility:hidden;
            -ms-backface-visibility:hidden;
            -o-backface-visibility:hidden;
            backface-visibility:hidden;
        }
        .card .front {background:lightblue;}
        .card .back {
            background:lightgreen;

            -webkit-transform:rotateY(180deg);
            -moz-transform:rotateY(180deg);
            -ms-transform:rotateY(180deg);
            -o-transform:rotateY(180deg);
            transform:rotateY(180deg);
        }

<div id="ex3">
  <div class="card">
    <figure class="face front">1</figure>
    <figure class="face back">2</figure>
  </div>
</div>
