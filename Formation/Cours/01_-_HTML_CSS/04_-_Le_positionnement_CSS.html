Flux normal

La propriété position

La position relative

Par défaut, les navigateurs affichent les boîtes issues du document html dans l’ordre du flux normal. Le positionnement relatif permet d'inscrire un contenu en flux normal, puis de le décaler horizontalement et/ou verticalement. Le contenu suivant n'est pas affecté par ce déplacement, qui peut donc entraîner des chevauchements.

Lorem Boite de type en-ligne en position relative ipsum dolor.


.s_1{
    background-color:lightblue;
    position:relative;
    bottom:5px;
    left:10px;
}

<p>Lorem <span class="s_1">Boite de type en-ligne en position relative</span> ipsum dolor.</p>

La position absolue

La position absolue (position:absolute) permet de ne pas dépendre de l'ordre dans lequel les éléments HTML sont déclarés dans le code, contrairement aux flottants que nous verrons plus tard.

Rappelons un point important concernant ce mode de positionnement : un élément positionné en absolu se réfère non pas à son parent direct, mais au premier ancêtre positionné qu'il rencontre.

Boite de type bloc en position absolue

.c_1{
    background-color:lightblue;
    height:500px;
    width:500px;
    position:relative;
}
.c_2{
    background-color:lightgreen;
    height:250px;
    width:250px;
    position:absolute;
    top:50px;
    left:50px;
}

<div class="c_1">
    <div class="c_2">Boite de type bloc en position absolue</div>
</div>

La position fixe

Le positionnement fixe s'apparente au positionnement absolu, à l'exception des points suivants:

  • Lorsque le positionnement est précisé (top, right, …), l'élément est toujours positionné par rapport à la fenêtre du navigateur
  • L'élément est fixé à un endroit et ne pourra se mouvoir, même lors de la présence d'une barre de défilement. En d'autres termes, la position intiale est fixée au chargement de la page, le fait qu'une éventuelle scrollbar puisse être utilisée n'a aucune influence sur le positionnement de l'élément: il ne bouge plus de la position initialement définie.
Boite de type bloc en position fixe

.c_1{
    background-color:lightblue;
    position:fixed;
    bottom:10px;
    right:0;
}

<div class="c_1">Boite de type bloc en position fixe</div>

La propriété flottante

Rôle & détournement des flottants

Une boîte flottante est retirée du flux normal, et placée le plus à droite (float: right) ou le plus à gauche (float: left) possible dans son conteneur. Le contenu suivant cette boîte flottante s'écoule le long de celle-ci, dans l'espace laissé libre.

Boite de type bloc en float left
Boite de type bloc en float right

.c_1{
    background-color:lightblue;
    float:left;
    height:150px;
    width:50%;
}
.c_2{
    background-color:lightgreen;
    float:right;
    height:150px;
    width:50%;
}

<div class="c_1">Boite de type bloc en float left</div>
<div class="c_2">Boite de type bloc en float right</div>

Nettoyer les flottants

La propriété clear s'utilise conjoitement aux float et permet à un élément (qui peut être d'ailleurs lui-même flottant) de ne plus subir le comportement d'habillage dicté par un objet flottant qui le précède directement et, par conséquent, de se caler en-dessous de ce dernier. Le clear autorise un nettoyage des flottants exclusivement à gauche (clear:left), à droite (clear:right) ou les deux simultanément (clear:both).

Boite de type bloc en float left
Boite de type bloc en float left & clear
Boite de type bloc en float left

.c_1{
    background-color:lightblue;
    float:left;
    height:150px;
    width:33.3333333333%;
}
.c_2{
    background-color:lightgreen;
    clear:left;
    float:left;
    height:150px;
    width:33.3333333333%;
}
.c_3{
    background-color:magenta;
    float:left;
    height:150px;
    width:33.3333333333%;
}
