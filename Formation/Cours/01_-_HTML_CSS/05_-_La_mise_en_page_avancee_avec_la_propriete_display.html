Utilisation du bloc en-ligne

La valeur inline-block intervient en permettant de bénéficier à la fois de l'élément en tant que bloc — et par la même occasion de pouvoir spécifier une taille — ainsi qu'en tant qu'élément en ligne au niveau de la gestion des placements.

Rappelons un point important concernant ce mode de positionnement : le navigateur ajoute un espace entre nos éléments inline-bloc. Ce défaut peut être corrigé en appliquant un "font-size:0;" sur l'élément conteneur.
Attention néanmoins à bien réattribuer un font-size sur les blocs inline-bloc sinon leur contenu textuel ne s'affichera pas !

Si on compare l'utilisation du display:inline-bloc avec celle de la propriété de positionnement float, on remarque que le conteneur a conservé sa hauteur.

Application sur des éléments de type bloc

Boite de type bloc 1
Boite de type bloc 2

#inline-bloc{
    background-color:black;
    font-size:0;
}
    #inline-bloc .c_1{
        background-color:lightblue;
        display:inline-block;
        font-size:12px;
        height:100px;
        width:50%;
    }
    #inline-bloc .c_2{
        background-color:lightgreen;
        display:inline-block;
        font-size:12px;
        height:150px;
        width:50%;
    }

<div id="inline-bloc">
    <div class="c_1">Boite de type bloc 1</div>
    <div class="c_2">Boite de type bloc 2</div>
</div>

L'alignement vertical

En temps qu'éléments possédant les caractéristiques du type en-ligne, les blocs en inline-bloc adjacents peuvent s'aligner entre eux.

Boite de type bloc 1
Boite de type bloc 2

#inline-bloc{
    background-color:black;
    font-size:0;
}
    #inline-bloc.valign .c_1{
        background-color:lightblue;
        display:inline-block;
        font-size:12px;
        height:100px;
        vertical-align:middle;
        width:50%;
    }
    #inline-bloc.valign .c_2{
        background-color:lightgreen;
        display:inline-block;
        font-size:12px;
        height:150px;
        vertical-align:middle;
        width:50%;
    }

<div id="inline-bloc" class="valign">
    <div class="c_1">Boite de type bloc 1</div>
    <div class="c_2">Boite de type bloc 2</div>
</div>
